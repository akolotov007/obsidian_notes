	- created by Adobe

## Sharing the printer
- ##### printer share
	- printer connected to computer
	- computer shares the printer
	- computer needs to be running
- ##### Print Server
	- print directly to the printer

## Configuration Settings
- ##### duplex
	- save paper
	- print on both sides of the page wo/ manually flipping over the paper 
	- not all printers can do this
- ##### Orientation
	- portrait vs landscape
		- printer changes how it prints
- ##### Tray Settings
	- printers can have multiple trays
	- plain paper, letterhead
- ##### Quality
	- resolution 
	- color, greyscale, color saving

## Printer Security
- ##### User Authentication 
	- everyone can print
	- set rights and permissions
	- printing vs managing the printer
- ##### Badging
	- authenticate when using a printer
	- your job doesn't print until you use your employee badge
- ##### Audit Logs
	- cost management
	- security monitoring
	- event viewer / system events
		- may be built into the printer or print server
- ##### Secured Prints
	- printer must support secure printing 
	- define a passcode for a person
	- use the passcode at the printer
